Coding Components & Feasability Report

Features I Plan to Include:

  • Click and drag companies, parks, museums, stores, and other buildings and infrastructure onto a grid type layout
  • “i” icon under each to help users understand what they are and what they do
  • “About This Project” with information about the project and why it was created
  • A box on the side showing how walkable a city that the user created is
  • A display box on the bottom showing what users have placed
  • Rotating gradient background in the back

Some libraries I have been looking into for this project include Typed.js, Draggable JS, Interact.js, and Granim.js.

For these libraries, the only library I have worked with already is Granim.js, so I have to get familiar with the other three libraries. In order to do so, I think I will work on making mini files for all of them and try to do something with them to see if it works properly. With that, I will decide if I like using interact.js or Draggable js more, seeing if only one is enough to get my project to work.
