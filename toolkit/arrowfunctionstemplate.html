Changing A Function To An Arrow Function

In this function, it checks the time of a video, and displays 2 lines of text depending on if its within 4 seconds. From the Blockbuster Video studio!

Named Function

            function checkTime() {
                if (1 < video.currentTime && video.currentTime < 4) {
                    line1.className = "showing";
                    line2.className = "showing";
                } else {
                    line1.className = "hidden";
                    line2.className= "hidden";
                }
            }

Function Expression

            const checkTime = function() {
                if (1 < video.currentTime && video.currentTime < 4) {
                    line1.className = "showing";
                    line2.className = "showing";
                } else {
                    line1.className = "hidden";
                    line2.className= "hidden";
                }
            }

Arrow Function

            const checkTime = () => {
                if (1 < video.currentTime && video.currentTime < 4) {
                    line1.className = "showing";
                    line2.className = "showing";
                } else {
                    line1.className = "hidden";
                    line2.className= "hidden";
                }
            }
